## Fiscal.ai (formerly FinChat) API Free Trial

Currently we offer a free trial which includes a subset of companies and rate limit.

To increase limits, please contact <a href="https://fiscal.ai/contact-sales/">Our Sales Team</a>

## Rate Limit
For the free plan, we support:

<Info>
**API Calls: 250 calls/day**
</Info>

### What happens when I hit the limit?
- API returns `403 Forbidden` with error message "Rate limit exceeded"
- Your daily limit resets at midnight UTC
- To continue using the API, upgrade to a paid plan or wait for the reset

### Monitoring Your Usage
- Each API response includes rate limit headers showing your remaining calls
- Track your usage through the organization dashboard
