## Quick Examples

Get started with these common API calls:

### Get Company Profile
```bash
curl "https://api.fiscal.ai/v1/company/profile?companyKey=NASDAQ_MSFT&apiKey=YOUR_API_KEY"
```

### Get Latest Financial Ratios
```bash
curl "https://api.fiscal.ai/v1/company/ratios?companyKey=NASDAQ_MSFT&apiKey=YOUR_API_KEY"
```

### Get Quarterly Earnings (Standardized)
```bash
curl "https://api.fiscal.ai/v1/company/financials/income-statement/standardized?companyKey=NASDAQ_MSFT&periodType=quarterly&apiKey=YOUR_API_KEY"
```

<Info>
Replace `YOUR_API_KEY` with your actual API key from the [getting started guide](/guides/getting-started).
</Info>

  </Card>

**Key Insight**: Standardized data isn't separate from as-reported data—it's as-reported data **plus the mapping layer**. You see both the final number AND how it was calculated.
